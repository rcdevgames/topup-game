4. API Usage Examples

JavaScript Fetch Example:

// 1. Get CSRF Token
const tokenResponse = await fetch('/api/csrf');
const tokenData = await tokenResponse.json();
const csrfToken = tokenData.data.token;

// 2. Use token in POST request
const response = await fetch('/api/auth/register', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'X-CSRF-Token': csrfToken  // Required for all POST/PUT/PATCH
    },
    body: JSON.stringify({
        phone: '+628123456789',
        password: 'password123'
    })
});

cURL Example:

# 1. Get CSRF Token
curl -X GET http://localhost:8080/api/csrf

# 2. Use token in POST request
curl -X POST http://localhost:8080/api/auth/register \
  -H "Content-Type: application/json" \
  -H "X-CSRF-Token: YOUR_TOKEN_HERE" \
  -d '{"phone": "+628123456789", "password": "password123"}'
